# Fournir des informations sur votre entreprise à Google

 
 
 
 Vous pouvez améliorer l'apparence et la couverture de votre site Web officiel, ainsi que votre présence dans les résultats de recherche en commençant par les définir auprès de Google. De cette façon, les utilisateurs peuvent plus facilement reconnaître votre site officiel et accéder aux informations que vous fournissez lors de leurs recherches. 

 Google propose différentes méthodes pour vous aider à fournir des informations clés sur votre entreprise afin qu'elles apparaissent dans les résultats de recherche. Cette section explique comment procéder pour que le lieu de votre établissement, votre site officiel et des informations sur le contenu soient disponibles dans les résultats de recherche, les fiches info Google et Google Maps. 

 

# Revendiquer un établissement local 

 Gérez l'affichage de votre établissement sur Google Maps et dans la recherche Google en revendiquant votre fiche d'établissement . Une fois que vous avez confirmé que vous êtes le propriétaire d'une fiche, vous pouvez fournir ou modifier l'adresse, les coordonnées et le type de l’établissement, et ajouter des photos.
 Ainsi, les informations relatives à votre établissement local s'affichent dans Google Maps et la fiche info Google. 

 

# Enregistrer votre site Web via la Search Console

 La première étape pour enregistrer votre site Web comme site officiel consiste à en valider la propriété dans la Search Console . Ce processus permet de valider les propriétaires et les opérateurs du site. Après avoir validé votre site Web,
 vous pouvez accéder à la Search Console pour identifier et contrôler la manière dont Google affiche les informations concernant votre
 site.

 

# Mettre à jour la fiche info Google de votre site

 Les algorithmes Google identifient les informations qui sont accessibles au public sur le Web telles que le nom de votre site, les coordonnées de l'entreprise et ses profils sur les réseaux sociaux. Vous pouvez mettre à jour
 ces informations ou en fournir d'autres afin d'accroître la portée de votre site dans les résultats de recherche et de mieux le faire
 connaître. Si vous êtes un représentant officiel agréé du site, vous pouvez
 mettre à jour la fiche info Google afin de remplacer les informations identifiées automatiquement par Google.

 

# Ajouter des données structurées

 Le moteur de recherche de Google met tout en œuvre pour comprendre le contenu d'une page. Dans cette optique, vous pouvez nous fournir des informations claires sur la signification d'une page en y ajoutant des données structurées.
 Ces données structurées représentent un format normalisé permettant de fournir des informations sur une page et de classer son contenu. En savoir plus sur les données structurées 

 Voici quelques fonctionnalités de données structurées dont tous les sites Web peuvent bénéficier :

 
 
- Logo du site  : vous pouvez spécifier l'image que Google doit afficher en tant que logo de votre établissement dans les résultats de recherche et dans votre fiche info. Pour ce faire, ajoutez à votre site Web officiel des données structurées Organization afin d'identifier l'emplacement de votre logo de prédilection . 
 
- Fil d'Ariane  : un fil d'Ariane sur une page indique la position de celle-ci dans la hiérarchie du site. Un utilisateur peut parcourir toute la hiérarchie du site, un niveau à la fois, en commençant par le dernier fil d'Ariane. Pour faciliter l'interprétation de votre fil d'Ariane par Google, vous pouvez ajouter à votre site des données structurées spécifiques à ce type de contenu . 
 

 

# Mettre en avant les modes de contact de votre service client

 Les clients recherchent souvent comment contacter une entreprise. C'est pourquoi Google s'efforce de présenter les meilleures informations disponibles pour les aider de différentes manières . Pour mettre en avant les modes de contact de votre service client dans la recherche Google, veillez à respecter plusieurs bonnes pratiques . Celles-ci vous permettront de vous assurer que les informations fournies pour votre entreprise ou votre service sont exactes.

 

# Dépannage

 Si vous rencontrez des problèmes concernant les informations de votre site, suivez les conseils de dépannage ci-dessous :

 
 
- Apportez les modifications suggérées à votre fiche info Google. 
 Si vous avez attendu au moins une semaine depuis la dernière exploration de votre page balisée par Google, signalez les informations qui vous semblent erronées dans une fiche info Google. Pour ce faire, cliquez sur le lien Commentaires situé au bas de la fiche.
 Si vous êtes une autorité reconnue, ou si vous possédez le site ou l'entité représenté dans la fiche info Google, d'autres options vous sont proposées pour apporter des modifications spécifiques.

 En savoir plus sur la marche à suivre pour mettre à jour une fiche info Google 
 
 
- Utilisez le test des résultats enrichis pour vérifier le balisage . Comptez un délai d'une semaine pour que notre système identifie et mette à jour les détails de votre site dans les résultats de recherche.
